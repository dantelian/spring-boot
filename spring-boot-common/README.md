# spring-boot-common

## office tool

- apache.poi wold

    - apache.poi生成wold文件，支持文本、表格(支持行、列合并)、图片
    
        示例：OfficeWordService>buildApacheWord
        
    - poi-tl 根据模板生成wold, 支持文本、图片、表格
    
        示例：OfficeWordService>buildApacheWord
        
        > poi-tl 基于Apache POI的Word模板引擎，较简单
